# 안녕하세요!

이것은 당신의 새로운 Next.js 블로그의 첫 번째 포스트입니다.

## 특징

- 📝 Markdown 형식으로 쉽게 글 작성
- 🎨 Tailwind CSS로 예쁘게 스타일링
- 📱 모바일 반응형 디자인
- ⚡ 빠른 성능

## 시작하기

`src/posts` 폴더에 `.md` 파일을 추가하면 자동으로 블로그에 표시됩니다.

파일명은 URL 경로가 됩니다. 예: `my-first-post.md` → `/blog/my-first-post`

즐거운 블로깅되세요! 🎉
